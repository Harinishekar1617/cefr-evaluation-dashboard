================================================================================

GEMINI API SETUP (RECOMMENDED - Fastest & Cheapest)
================================================================================

1. GET API KEY:
   Visit: https://aistudio.google.com/app/apikeys
   Click: Create API Key
   Copy: Your API key (starts with AIza...)

2. STORE API KEY:
   Method A - Using .env file (Recommended):
   $ cd "/Users/hshekar/CEFR Evaluation"
   $ echo "GEMINI_API_KEY=YOUR_KEY_HERE" > .env

   Method B - Direct in config.py:
   Edit: cefr_pipeline/config.py
   Change: GEMINI_API_KEY = "YOUR_KEY_HERE"

3. INSTALL LIBRARY:
   $ pip install google-generativeai

4. CONFIGURE:
   Edit: cefr_pipeline/config.py
   Set: API_PROVIDER = "gemini"

5. RUN:
   $ python main.py

6. RESULTS:
   Check: output/cefr_results_all_students_gemini.xlsx

================================================================================
SWITCHING TO OTHER PROVIDERS
================================================================================

FOR OPENAI:
-----------
1. Get key: https://platform.openai.com/api-keys
2. Add to .env: OPENAI_API_KEY=sk-...
3. In config.py: API_PROVIDER = "openai"
4. Run: python main.py

FOR OPENROUTER:
---------------
1. Get key: https://openrouter.ai/keys
2. Add to .env: OPENROUTER_API_KEY=sk-or-...
3. In config.py: API_PROVIDER = "openrouter"
4. Run: python main.py

================================================================================
TROUBLESHOOTING
================================================================================

Q: "google-generativeai not installed"
A: pip install google-generativeai

Q: "Invalid API Key"
A: • Verify key format: AIza... for Gemini
   • Check it's active at aistudio.google.com/app/apikeys
   • No extra spaces in .env file

Q: "Rate limit exceeded"
A: Wait a few minutes and try again. Free tier has limits.

Q: "Connection timeout"
A: Check internet connection. Try again in a few moments.
